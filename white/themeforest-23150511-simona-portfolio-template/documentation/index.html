C) CSS Files and Structure - top

Styles

We're using 7 core CSS files in this template. we have separated styles for improving convenience to edit this template. If you're going to use this template in production we recommend you merge all stylesheets in one file (for decreasing http queries).

Lets take a look closely at our stylesheet files.

bootstrap.min.css

This Stylesheet contains responsive grid system, default element style based on bootstrap. It is bootstrap framework main css file.

font-awesome.css

Base styles for FontAwesome icon fonts.

fullpage.min.css

Use for scrolling.

owl.carousel.css

Use for sliders

animate.css

Use for blocks animation

style.css

Main css file where you can do own customization
